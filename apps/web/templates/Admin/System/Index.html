　システム
システム管理は、BlueDrive 自体の操作やSyLフレームワークのプロジェクトに関わる操作を行います。

　ユーザー

ユーザー管理

BlueDriveを利用するユーザーを管理します。

グループ管理

BlueDriveを利用するユーザーが所属するグループを管理します。

アクセス範囲管理

BlueDriveを利用するユーザーが、アクセスするURLの範囲を管理します。

　ファイル

ファイル保存領域管理

サーバー上にあるファイルの保存領域の定義を管理します。

　SyL

ログ

管理対象のSyLフレームワークのログを収集した結果を確認します。
